# 🛒 Tech Store AI Bot

Telegram-бот-консультант для офлайн-магазина техники. Принимает обычные вопросы покупателей («сколько стоит iPhone 17 Pro», «есть кнопочный телефон до 3000?», «посоветуй смартфон до 25 тысяч») и отвечает на них **на полностью локальной LLM** — ничего не уходит в OpenAI / Google / Anthropic.

## ✨ Что умеет

- Понимает естественный язык — клиенту не нужно знать команды
- Работает на **локальной LLM** (Qwen 2.5 14B Instruct через LM Studio) — нулевая стоимость, приватность данных
- Знает только реальный каталог магазина — не выдумывает цены и модели
- Подсвечивает релевантные товары динамически: на запрос «Redmi A5» в промпт LLM подставляются только подходящие позиции, остальной каталог — однострочником
- Отвечает в человечном тоне: 1–3 короткие фразы, без списков и канцеляризмов
- Не сливает клиенту служебные ID, теги, остатки на складе и категории из базы

## 🧰 Стек

- **Python 3.13** + `asyncio`
- **aiogram 3** — Telegram bot framework
- **OpenAI Python SDK** — общение с LM Studio через OpenAI-совместимый API
- **LM Studio** + **Qwen 2.5 14B Instruct (Q4_K_M)** — локальная LLM
- **python-dotenv** — конфиги через `.env`
- **dataclasses + JSON** — каталог как простая структура без БД (так и должно быть на 70 SKU)

Запускается на MacBook Pro M1 16 GB. Прожитой памяти — около 9–10 GB при загруженной модели.

## 🚀 Установка

### 1. Зависимости

Нужен Python 3.11+. Macbook рекомендуется, потому что Qwen 2.5 14B на M1/M2/M3 идёт через Metal acceleration в LM Studio.

```bash
git clone https://github.com/<твой-юзер>/ai-bot.git
cd ai-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. LM Studio

1. Скачай LM Studio с [lmstudio.ai](https://lmstudio.ai)
2. В LM Studio найди и скачай `qwen2.5-14b-instruct` (Q4_K_M, около 9 GB)
3. Открой Developer → Local Server, нажми **Start Server**
4. Загрузи модель с настройками:
   - Context Length: **4096** (на 16 GB Mac не больше)
   - Max Concurrent Predictions: **1**
   - GPU Offload: max

API будет доступно на `http://localhost:1234/v1`.

### 3. Telegram-бот

1. Открой [@BotFather](https://t.me/BotFather), `/newbot` — получи токен
2. Скопируй конфиг и подставь свой токен:
   ```bash
   cp .env.example .env
   # отредактируй .env: TELEGRAM_TOKEN=...
   ```

### 4. Запуск

```bash
./venv/bin/python bot.py
```

В терминале появится `✅ Бот запущен!`. Открывай чат с ботом в Telegram и пиши.

## 📦 Каталог

Лежит в `data/products.json`. Структура одной записи:

```json
{
  "id": "phn-ap-001",
  "name": "iPhone 17 Pro Max 256GB",
  "category": "смартфон",
  "price_rub": 139990,
  "stock": 4,
  "specs": "Apple A19 Pro, 6.9\" ProMotion OLED, тройная 48MP, USB-C, титан",
  "tags": ["ios", "флагман", "камера", "apple", "iphone"]
}
```

Поменял каталог → перезапусти бота (`Ctrl+C`, потом `python bot.py`). Системный промпт и каталог кэшируются на старте через `@lru_cache`.

## 🧠 Как устроен промпт

На каждый запрос пользователя бот собирает три блока:

1. **Базовый промпт** (`prompts/system_prompt.txt`) — правила тона и поведения. Например, «не предлагать альтернативу, если клиент сам не попросил» и «никогда не называть количество штук на складе».
2. **Компактный каталог** — все 70+ товаров одной строкой каждый: `iPhone 17 Pro Max 256GB — 139990₽ (есть)`. Это даёт модели общую картину ассортимента без лишних токенов.
3. **Детали по релевантным товарам** — наивный поиск по словам из запроса находит топ-3 совпадения, и они вставляются в промпт уже с полными specs. Так модель отвечает по сути, а не пересказывает половину каталога.

Итого — около 1500 input-токенов на каждый запрос вместо 4000+ при «положим весь каталог в промпт». Скорость ответа на M1 Pro 16 GB — 20–60 секунд (узкое место — prompt processing 14B-модели).

## 🐛 Troubleshooting

**Бот не отвечает, в логах `Retrying request to /chat/completions`** — модель всё ещё прожёвывает промпт. На 14B + 16 GB первый ответ идёт медленно, но дойдёт. В `.env` стоит `LM_STUDIO_TIMEOUT=600`.

**`LM Studio недоступна`** — открой LM Studio, проверь Status: Running и что модель загружена.

**Бот говорит цены в долларах / старыми моделями** — ты не перезапустил бота после правки каталога. `Ctrl+C` и `python bot.py` заново.

**ОЗУ под завязку** — выгрузи модель, загрузи заново с Context Length 4096 и Max Concurrent Predictions 1. Сохраняется до 3 GB.
